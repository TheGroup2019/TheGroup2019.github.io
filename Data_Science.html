Data Science

Scratch Python Web Development Robotics

You're still in python, but now you will learn how to be a basic Data Scientist. You will use bar graphs, histograms, any type of graph you want to demonstrate data.

Survey


image source=https://www.itnews.com.au/news/win-200-in-the-2019-itnews-readership-survey-519456

Here you create a survey about anything you'd like. It's quite easy actually. But the goal isn't just to have a working survey. You must be able to collect to the answer the user provides and to display it as a graph of some type. Whether you show it as a word document or a list, it has to display all answewrs from everyone who took your survey. An example:
Name: Nancy. Age: 16. Grade Level: Junior.
Name: Melissa. Age: 17. Grade Level: Senior.
Name: Veronica. Age: 18. Grade Level: Senior.
Name: Melissa. Age: 17. Grade Level: Junior.

Data Visualization & Data Scientist for a day.


image source=https://www.slideshare.net/CarlAnderson4/setting-up-data-science-for-success-the-data-layer

Since both projects are really similar, I decided to explain it under one heading. All you need to do is present visual data for both.
Data Visualization: Here with the twitter data set you have, you'll be showing some of the common words people use for twitter(not counting words such as: the, it, to, and, etc). Pretty cool project, because you can actually see your work on a chart. As appealing as it sounds to me, however, I'm not the Data Science type of gal.
Data Scientist for the Day: This project is slightly harder, but you're pretty much doing the same thing as before, but with a new dataset, and presenting new data. For example: your data could the type of music kids listen to this year, between the ages 0f 5-10. You could present it as a line graph, histogram, bubble chart, These projects are cool and fun, but I'm more of a web developer.
